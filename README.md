# Dungeons and Dragons

Campaigns and oneshots for D&D sessions.

## Structure

```
campaigns/       — Full campaign arcs (each campaign gets its own subfolder)
oneshots/        — Standalone one-session adventures
resources/       — Shared reference material
  maps/          — Battle maps and world maps
  npcs/          — NPC profiles and stat blocks
  items/         — Magic items, loot tables
  lore/          — World lore and setting notes
  rules/         — D&D rules references and homebrew rules
```

## How to use

- Create a new folder under `campaigns/` or `oneshots/` for each adventure
- Use markdown files for session notes, plot outlines, and encounter design
- Store shared resources (recurring NPCs, world lore) in `resources/`
